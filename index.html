----SELECTION : WEB OR APP----

When you are in your second year of college, you have to choose between web development or application development.Until you finish BCA, you only know about the field you have chosen.If you have chosen web development field then you are given knowledge of HTML CSS and javascript etc. for front web development and in addition to this you are also given knowledge about mysql database, asp.net, php etc. for backend web development.Nowadays, the college also provides information about new frameworks for CSS and javascript. Such as Bootstrap, Tailwind, Skeleton for CSS and ReactJs, Angular etc. for JavaScript.

When you are in your second year of college, you have to choose between web development or application development.Until you finish BCA, you only know about the field you have chosen.If you have chosen the application development field, you are taught programming languages like Java and Kotlin.In addition, deep knowledge is also given about Android Studio, Flutter, react native etc.There is also knowledge about hybrid and native application.Also required for application development, all of which are provided by the Knowledge College.
